4. Pravila headera u Excelu

  • Prvi red = nazivi kolona (header)
  • Podaci od reda 2
  • Može srpski: id dela, naziv dela, kom za kontrolu n
  • Zvezdica na kraju (id dela*) — ignoriše se
  • Đ/č/š u headerima — automatski se mapira
  • DA / NE za aktivan, aktivna, aktivno

8. Uvoz u bazu — korak po korak

A) Atributivni šifrarnik

  1. Izmeni SPC_master_atributivne.xlsx u excel-rad\
  2. Admin panel → Atributivne → ADMIN → Uvezi iz Excela
  3. Proveri preview (broj redova po tabu) → Uvezi u Supabase

Alternativa: izvezi CSV u docs\ → npm run import:docs

B) Merljive granice

  1. Izmeni SPC_merljive.xlsx — tab karakteristike_merljive
  2. Modul Merljive → Modul 2 → tab EXCEL → Uvezi
  3. Auto-sync puni SOP, delovi, RN iz karakteristika

C) Provera posle uvoza

  1. Aplikacija → unesi ID dela → učitava se crtež i granice
  2. Atributivne → dropdown grešaka
  3. Vozilo → dijagram + kategorije po zoni

10. Radni tok (praksa)

1. Novi deo     → delovi + greske_katalog (atrib.) ILI karakteristike_merljive (merlj.)
2. Granice      → samo karakteristike_merljive (inženjer)
3. Novi radnik  → radnici + Auth + Admin Lozinka
4. Novi RN      → radni_nalozi ili ERP CSV
5. Uvoz         → Admin / Merljive EXCEL
6. Svaki dan    → operateri u aplikaciji (NE Excel)
