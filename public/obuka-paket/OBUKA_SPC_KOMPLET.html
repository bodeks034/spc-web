Kompletna obuka aplikacije · za sve uloge · SPC Web

2. Modul 1 — rad na liniji

Pre ulaska — Kontrolna lista

Prvi put u smeni (Modul 1) potvrđuješ stavke kontrolne liste (osim admina). Bez toga unos je blokiran. Stavke se uređuju u Modul 0 → Osnovno → Kontrolna lista.

Zlatna pravila (operater / kontrolor)

PraviloZašto
Prvo POKA-YOKE, pa unosManje grešaka „pogrešan deo / merilo“
Uvek Snimi / ZapišiDraft nije u bazi — LOG ga ne vidi
Ne menjaš granice LSL / USLTo radi kvalitet u šifrarniku
Ne menjaš greške u šifrarnikuSamo biraš iz liste
Crvena blokada = staniSPC alarm / karantin / kalibracija — zovi kvalitet

Atributivne — OK / NOK

Podaci: tabela merenja_varijabilna. LOG = pregled (filter datum/smena; offline 📶 na vrhu).

2b. Linija — LOG, MOMENT, FAI, offline

LOG

Pregled unosa iz baze. Filter: datum, smena, dugme Danas. Stavke sa 📶 još nisu poslati (offline red) — nestaju posle sinhronizacije.

MOMENT (digitalni moment ključ)

Samo u merljivom digitalnom režimu. Operater prati sekvencu JOB-ova (Nm, redosled). JOBove podešava kvalitet u Modul 0 → Moment ključ → Šifrarnik JOB.

FAI (First Article Inspection)

ŠtaPravilo
Tab FAIKontrolor (i kvalitet+) u merljivim
Odobrenje OKKontrolor sme (bez NOK)
Odobrenje sa NOKSamo kvalitet / šef / admin

Offline

Ako nema mreže, unos ide u lokalni red (IndexedDB). Kad se mreža vrati — automatski se šalje u bazu. Toasts: „Offline: … u redu“. Admin vidi status offline synca u Admin panelu.

Barkod

KakoRezultat
Sken kamerom ili HID skenerom na UNOS-uBrži ID / RN
Etikete: Modul 0 → Alati → BarkodKvalitet štampa etikete
